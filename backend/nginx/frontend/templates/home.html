Welcome to MightyPong!

This website has been created for the final project at 42 Adelaide, ft_transcendence.
It’s a full-stack single page application made with HTML, CSS, JavaScript, and Ruby.

Here’s a full list of the modules that have been completed:

Backend Database
All database instances within the project use PostgreSQL.
Standard User Management
Secure registration and login, with access to a user profile.
Remote Players
It is possible to have two distant players on separate computers.
Another Game
Developed a new game to diversify the platform's offerings.
Game Customisation Options
Offer customisation features to enhance the gameplay experience.
Gameplay via CLI
Created a robust CLI that connects to the web application.
Server-Side Pong
Developed server-side logic for the Pong game + implemented an API.
Live Chat
Able to interact with other users and send direct messages.
